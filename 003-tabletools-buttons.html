TableTools Buttons

You can choose to use a subset of the buttons in the TableTools extension, or arrange some buttons in a collection. There are such examples on the DataTables website. We rewrite two simple examples in R below:

library(DT)
iris2 = head(iris, 20)
# only show the Copy and Print buttons
datatable(
  iris2,
  extensions = 'TableTools', options = list(
    dom = 'T<"clear">lfrtip',
    tableTools = list(sSwfPath = copySWF(), aButtons = c('copy', 'print'))
  )
)

# put CSV, XLS, and PDF in a collection
datatable(
  iris2,
  extensions = 'TableTools', options = list(
    dom = 'T<"clear">lfrtip',
    tableTools = list(
      sSwfPath = copySWF(),
      aButtons = list('copy', 'print', list(
        sExtends = 'collection',
        sButtonText = 'Save',
        aButtons = c('csv', 'xls')
      )))
  )
)

We mentioned that the PDF button is disabled by default if you use DT::copySWF(). You can enable it using copySWF(pdf = TRUE):

datatable(iris2, extensions = 'TableTools', options = list(
    dom = 'T<"clear">lfrtip',
    tableTools = list(sSwfPath = copySWF(pdf = TRUE))
))
